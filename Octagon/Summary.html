Заключение

Что можно посмотреть на Web-страницах других моих сайтов:

На сайте Three.js и геометрия все 16 моделей многогранников расчитаны с помощью библиотеки WebGeometry (так же как и на текущем сайте), но отображены они на холст с помощью ThreeJS.

На сайте Diamond Cuts модели 7 многогранников также расчитаны спомощью библиотеки WebGeometry и отображены на холст с помощью ThreeJS. При этом используется несколько типов шейдеров и кубические текстуры.

На сайте Diamond Cuts Collection можно найти множество моделей различных многогранников. При определении координат вершин всех этих моделей координаты предварительно расчитывались на Visual C++ и их значения записывались в файлы. Эти файлы с сохраненными в них значениями координат используются в программах отображения моделей на экран (ThreeJS и API WebGL).

  На страницах Shine on You Crazy Diamond ! вышеназванного сайта можно под музыку двигаться по сцене и рассматривать модели огранок. Управление движением - мышь и клавиатура как в компьютерных играх. Вывод на экран осуществляется при помощи ThreeJS.

  По ссылке Pages можно перейти на страницы сайта с 36 моделями различных огранок. Вывод на экран на этих страницах осуществляется при помощи чистого API WebGL.

  По ссылке Diamond Cuts with Dispersion Light можно перейти на страницы сайта с на которых модели отображены при помощи шейдеров и кубических текстур. Вывод на экран моделей с этих страниц осуществляется при помощи ThreeJS.
